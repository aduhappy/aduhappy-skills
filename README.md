=== 第一部分：AI Agent Skills 功能简述备忘录 ===
【分类一：学术科研与论文写作】
1. Norman-bury/research-writing-skill（约 2000 星标）：科研写作助手。主打规范流程，拒绝AI盲目扩写和乱编参考文献，支持解析 LaTeX 模板。
2. Yuan1z0825/nature-skills（约 3200 星标）：Nature 级学术表达与绘图。专门用于将语言润色成顶级期刊风格，并辅助编写符合顶刊要求的科研图表代码。
3. K-Dense-AI/scientific-agent-skills（约 25800 星标）：科学智能体大礼包。内置 130 多种技能，能直接调用生物、地理等 70 多个权威硬核科学数据库。
4. Imbad0202/academic-research-skills（约 21200 星标）：学术研究全流程管理器。提供从梳理大纲到定稿的一条龙服务，能学习你的写作风格，并有防泄密保护。
5. Rimagination/scansci-pdf（新锐项目）：科研 PDF 扫描解析器。能精准读取普通 AI 难以处理的复杂论文（包含复杂公式、双栏排版和图表数据）。
【分类二：文本润色与去 AI 痕迹】
6. op7418/Humanizer-zh（新锐项目）：中文 AI 写作去痕工具。识别并消除常见的“AI套话”，让文章逻辑和表达更符合真实人类的中文书写习惯。
【分类三：开发辅助与知识产权】
7. Fokkyp/SoftwareCopyright-Skill（小众实用工具）：软件著作权材料生成器。国内开发者必备，一键自动抽取项目前后 30 页代码，并生成繁琐的软著申请说明书。
【分类四：对话增强与长文本记忆】
8. Rimagination/ChatMem（基础支撑项目）：智能体长短期记忆扩展。为 AI 建立结构化的后台记忆库，解决其在超长项目或几天连贯工作中“转头就忘”的上下文断层问题。
